# Multi-Endpoint File Integrity Monitoring (FIM) Lab with Wazuh SIEM

## 📌 Project Overview
This project demonstrates the deployment of a containerized **Wazuh SIEM Open-Source Stack** (Manager, Indexer, and Dashboard) used to monitor file integrity across distinct operating system environments on a single host. 

The lab showcases real-time security monitoring capabilities across:
1. **Windows 11 Pro** (Host Endpoint)
2. **Ubuntu 26.04 LTS via WSL2** (Virtual Linux Endpoint)

By implementing **File Integrity Monitoring (FIM)**, the environment captures and logs real-time file operations (Creations, Modifications, Deletions) through specialized OS kernel subsystems (**Inotify** on Linux and **ReadDirectoryChangesW** on Windows).

---

## 🏗️ Lab Architecture & Topology
* **SIEM Core:** Wazuh v4.8.2 Stack deployed via Docker Compose.
* **Manager Node Network:** `192.168.1.158` (Host LAN Bridge).
* **Agent 001 (Windows Host):** Native monitoring of target storage path `C:\FIM-Test`.
* **Agent 004 (WSL2 Ubuntu):** Bridged NAT monitoring of target path `~/test`.
